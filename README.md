## рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃЊрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

### ­ЪЊЂ django_project/
рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃцрЃЮрЃџрЃЊрЃћрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА Django-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА.

- **settings.py** РђЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃА, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃА, template-рЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃА рЃЊрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃА.
- **urls.py** РђЊ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃа URL рЃЏрЃљрЃарЃерЃарЃБрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА pages рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА.
- **asgi.py** РђЊ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљрЃА asynchronous рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃќрЃћ.
- **wsgi.py** РђЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ production рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў Django рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ.
- **\_\_init\_\_.py** РђЊ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃЏ рЃцрЃЮрЃџрЃЊрЃћрЃарЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА Python package-рЃА.

### ­ЪЊЂ pages/
Django рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃќрЃћ.

- **views.py** РђЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА class-based view-рЃћрЃЉрЃА (`HomePageView` рЃЊрЃљ `AboutPageView`), рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃћрЃю рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА HTML рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃА.
- **urls.py** РђЊ рЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА URL рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА view-рЃћрЃЉрЃЌрЃљрЃю.
- **tests.py** РђЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃбрЃћрЃАрЃбрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЊрЃљ About рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА.
- **apps.py** РђЊ pages рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃў.
- **models.py** РђЊ рЃфрЃљрЃарЃўрЃћрЃџрЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ.
- **\_\_init\_\_.py** РђЊ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃЏ рЃцрЃЮрЃџрЃЊрЃћрЃарЃў рЃљрЃарЃўрЃА Python package.

### ­ЪЊЂ templates/
HTML рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃўрЃА рЃцрЃЮрЃџрЃЊрЃћрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃюрЃљрЃгрЃўрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.

- **base.html** РђЊ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў template, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃАрЃ«рЃЋрЃљ HTML рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃўрЃЌ рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю.
- **home.html** РђЊ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃўрЃА HTML рЃерЃљрЃЉрЃџрЃЮрЃюрЃў.
- **about.html** РђЊ About рЃњрЃЋрЃћрЃарЃЊрЃўрЃА HTML рЃерЃљрЃЉрЃџрЃЮрЃюрЃў.
